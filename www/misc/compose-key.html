using the compose key

decided to setup a compose key for myself, not because i need to type other languages, but just because i like being able to type unicode hearts and smileys etc ☺♥⑨

for river, just tossing these into env before the compositor starts is enough

# compose key
export XKB_DEFAULT_LAYOUT=us
export XKB_DEFAULT_OPTIONS=compose:ralt

to use: press the compose key, (alt in this case), then the key combination
